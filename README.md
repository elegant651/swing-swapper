# Swing-swapper

> a component for swapping erc20s on Uniswap (plus tokenlists + local forks of mainnet!)

---

## quickstart

```bash

yarn install

```

```bash

yarn start

```

- In a second terminal window run:

```bash

yarn fork

```

Notes:
- This widget uses [tokenlists](https://tokenlists.org/) to import the erc20s of your choice
