# Swing-swapper

> Cross-chain Uniswap with swing API to enable cross-chain swaps
[WIP]
---

## quickstart

```bash

yarn install

```

```bash

yarn start

```

- In a second terminal window run:

```bash

yarn fork

```

Notes:
- This widget uses [tokenlists](https://tokenlists.org/) to import the erc20s of your choice
